# The Petty Court (소소한 재판소) - Blueprint

## 🏛️ 프로젝트 개요
친구, 연인 간의 사소한 다툼을 AI 판사가 판결해주는 유머러스한 웹 서비스입니다. 사용자가 양측의 주장을 입력하면 엄격하면서도 재치 있는 판결을 내려줍니다.

## 🎨 디자인 및 스타일
- **테마**: 엄숙하지만 위트 있는 법정 테마
- **색상 팔레트**:
  - Primary: `#60463B` (진한 법정 나무색)
  - Secondary: `#856A5D` (부드러운 나무색)
  - Background: `#DFE0E2` (연한 그레이, 석재 느낌)
  - Accent: `#D4AF37` (금색 포인트)
- **UI 특징**: 
  - 입체감 있는 카드 레이아웃 (Deep Shadow)
  - 클래식한 타이포그래피 (명조체 계열 권장)
  - 모바일 반응형 레이아웃

## ✨ 주요 기능
1. **메인 화면 (Main Court)**
   - 원고(Plaintiff) 및 피고(Defendant) 주장 입력란
   - '판결 내리기' 버튼 (망치 효과 시각화)
2. **판결 화면 (Verdict Room)**
   - 로딩 상태: 유머러스한 문구 표시
   - 결과 카드: 승자 이름, 죄목, 판결문(3~4문장), 형량/벌칙
   - '다시 재판하기', '판결문 공유' 기능
3. **AI 판사 로직**
   - 페르소나: 솔로몬 + 개그콘서트 (엄숙 + 유머)
   - 모드: 
     - **API 모드**: 실제 AI 호출 (설정 시)
     - **데모 모드**: API 키가 없을 경우 사전 정의된 랜덤 판례 제공

## 📋 구현 단계
1. `blueprint.md` 작성 및 기본 구조 설계 (완료)
2. `index.html`: 기본 HTML 구조 및 웹 컴포넌트 마크업
3. `style.css`: 법정 테마 스타일링 및 애니메이션 적용
4. `main.js`: 판결 로직, 데모 모드, 공유 기능 구현
5. 검증 및 GitHub 배포
